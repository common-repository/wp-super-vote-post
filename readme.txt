Votação de Posts, com confirmação via email

== Description ==

Permite a votação em posts, com confirmação do voto por email, que possibilita a
exportação dos dados para uma futura auditoria/newsletter.

== Installation ==
* Faça o upload do plugin para o seu blog e depois basta ativar.
